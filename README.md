# 🍽️ Rest Easy

**Aplicació web multiplataforma per gestionar fàcilment el teu negoci de restauració**

## 📌 Descripció del projecte
GestióRestaurant és una aplicació web dissenyada per simplificar i optimitzar les tasques del dia a dia en negocis com restaurants, bars i cafeteries. Permet gestionar reserves, comandes, inventari, facturació, impressió automàtica de tiquets i molt més, tot des d'un únic lloc i accessible des de qualsevol dispositiu.

## ✨ Funcionalitats principals

- **Gestió de reserves:** calendari interactiu amb confirmació automàtica.
- **Carta digital i gestió de comandes:** gestió fàcil i intuïtiva de la carta i les comandes.
- **Impressió automàtica de tiquets:** configuració d'impressores múltiples (cuina, barra, caixa).
- **Gestió d'inventari:** control automàtic de l’estoc amb alertes.
- **Facturació integrada:** emissió de factures i tiquets digitals.
- **Gestió de personal:** assignació fàcil de tasques i torns.
- **Informes i analítiques:** estadístiques per analitzar rendiment i millorar les vendes.

## 🚀 Tecnologies utilitzades

- **Frontend:** React.js, Tailwind CSS, Axios.
- **Backend:** Node.js, Express.js.
- **Base de dades:** MongoDB o PostgreSQL.
- **Desplegament:** Vercel o AWS.

## ⚙️ Instal·lació i ús

1. Clona aquest repositori:

```bash
git clone https://github.com/el-teu-usuari/gestiorestaurant.git
```

2. Instal·la dependències:

```bash
npm install
```

3. Executa l'aplicació en local:

```bash
npm run dev
```
